협업 변수명명 규칙

변수 시작명
1. 띄어쓰기 = 언더바(_)
2. 전체 소문자
3. 사용위치_동사_명사
-> 애매할경우 조율
4. 특정단어는 길 경우 조율해서 줄여서 사용 (모든 곳에서 동일하게 줄여서 사용)

Ex) 
    login_set_id
    main_search_food


함수 && 클래스 명 작성 규칙
1. 첫단어 Func_ 
2. 띄어쓰기 = 언더바(_)
3. Func_사용위치_동사_명사
-> 애매할경우 조율
4. 첫글자이후 소문자로
5. 사용위치 조율해서 줄여서 사용

Ex) Func_register_set_account
    Func_login_get_account
    Func_main_search_igd
    Func_rfg_set_Igd
        (rfg -> 냉장고)
    

커밋 메시지의 7가지 규칙
      1. 제목과 본문을 빈 행으로 구분한다
      2. 제목을 50글자 내로 제한
      3. 제목 첫 글자는 대문자로 작성
      4. 제목 끝에 마침표 넣지 않기
      5. 제목은 명령문으로 사용하며 과거형을 사용하지 않는다
      6. 본문의 각 행은 72글자 내로 제한
      7. 어떻게 보다는 무엇과 왜를 설명한다
